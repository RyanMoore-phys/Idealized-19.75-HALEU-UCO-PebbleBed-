# Idealized-19.75-HALEU-UCO-PebbleBed-
Using OpenMC, I have modeled a 19.75% enriched HALEU UCO Pebblebed with roughly 13 thousand TRISO pellets. Using idealized conditions (reflective boundary BC's &amp; cold temperature), this project determines Fresh-fuel k-inf, spectrum, and reaction rate ratios. The project also includes a 3 Year Burnup curve and Isotope Inventory
